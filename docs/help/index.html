The fastest path to unstuck, in order:

  1. Tap the Course Companion bubble in the bottom-right of any page — it's an AI grounded in the course's own materials.
  2. Check the Glossary if a term is the blocker.
  3. Skim the FAQ — most evening blockers are there.
  4. If none of the above worked, book a virtual office-hours slot for the next day.
